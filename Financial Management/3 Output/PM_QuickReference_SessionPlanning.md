## FACILITATION TIPS

### **Managing Time** (90 min COA section will feel tight)
- Use timer for each sub-section (COA-1A through 1E)
- If running 5-10 min over on usage analysis (1B), that's OK â€” that's the most valuable section
- If running over on COA-3 (target structure design), ask: "Do we have enough to move forward, or do we need another session?"
- Be prepared to defer detailed discussion of Sections 2C, 2D, 3 to follow-up calls
